## 핵심 사실 (검증됨)

### 1. 일본 정월의 시간 구조

```
12월 13일       — 正月事始め (정월 준비 시작일)
12월 28일       — 정월 장식 거는 권장일 (29일·31일은 피함)
                  └ 29일 = "二重苦" (이중고)로 읽혀 불길
                  └ 31일 = "一夜飾り" (하룻밤 장식)로 신께 결례
1월 1일~3일    — 三が日 (산가니치, 정월 3일간)
1월 1일~7일    — 松の内 (松七日, 関東/東北/九州 기준)
1월 1일~15일   — 松の内 (関西/오키나와 기준)
1월 7일       — 七草の節句 (나나쿠사, 7가지 풀죽 먹는 날)
1월 11일      — 鏡開き (가가미비라키, 가가미모치 깨서 먹는 날)
1월 15일      — 小正月 (코쇼가츠)
1월 20일      — 二十日正月·骨正月 (지방에 따라)
```

**한국 학습자에게 강조할 포인트**: 한국 설날은 음력 1월 1일 (보통 양력 2월), 일본은 양력 1월 1일. 이 차이 때문에 **연말연시 분위기가 한국보다 훨씬 짧지만 강렬**.

### 2. 年神様 (도시가미사마) 개념

- 정월에 각 가정을 방문하는 신
- 풍년·무병식재를 가져다줌
- 조상신과 동일시되기도 함
- **모든 정월 풍습의 핵심**: 年神様를 맞이하고 떠나보내는 의례

### 3. おせち料理 — 사실 검증

**구조**:
- 重箱(주바코) = 찬합. 보통 3단 또는 5단
- 약 20종류 (홀수, 짝수 피함)
- **5가지 분류**: 祝い肴 / 口取り / 焼き物 / 酢の物 / 煮物

**祝い肴三種 (가장 핵심)**:
- **関東**: 黒豆 + 数の子 + 田作り
- **関西**: 黒豆 + 数の子 + たたきごぼう

**대표 음식 의미** (학습자에게 가장 흥미):

| 음식 | 의미 |
|---|---|
| 黒豆 (쿠로마메) | "まめに働く" (성실히 일하다) → 건강·근면. 주름 = 장수 상징 |
| 数の子 (카즈노코) | 청어알 → 자손번영. 한자 二親 (양친) 으로도 씀 |
| 田作り (타즈쿠리) | 멸치 비료가 풍년 만든 유래 → 五穀豊穣 |
| 伊達巻 (다테마키) | 두루마리 형태 → 학문 성취 |
| 栗きんとん (쿠리킨톤) | 황금색 → 금전운 |
| 海老 (에비) | 새우의 굽은 등 → 장수 |
| 昆布巻 (콘부마키) | 喜ぶ(요로코부) 발음 유사 → 기쁨 |

**왜 정월에 이걸 먹나?**:
- 三が日 동안 **불 사용 자제** (家事休業, 가지 휴업)
- 보존 가능한 음식 중심
- 神様에게 失礼하지 않게 + 주부에게 휴식

**현대 트렌드**:
- 직접 만들지 않고 백화점·편의점에서 예약 (`おせち通販`)
- 가격 ¥10,000~¥30,000 (3-4인 기준)
- 외국인 거주자도 이세탄·다카시마야 백화점에서 11-12월에 예약 가능

### 4. お年玉 (오토시다마) — 2026년 기준 상세

**금액 상세 (年齢別)**:

| 연령 | 평균 금액 |
|---|---|
| 미취학아동 | 1,000~3,000엔 (1,000엔 중심) |
| 초등학교 저학년 | 2,000~3,000엔 |
| 초등학교 고학년 | 3,000~5,000엔 (4,001~5,000엔 가장 많음) |
| 중학생 | 5,000엔 (대부분 일률) |
| 고등학생 | 5,000~10,000엔 |
| 대학생/18세+ | 10,000엔 전후 (또는 종료) |

**관계별 금액**:
- 본인 자녀: 1세 단위로 책정
- 조카: 자녀와 동일 또는 약간 낮게
- 친구 자녀: 1,000~3,000엔 정도 (잘 모르는 경우)

**ぽち袋 (포치부쿠로) 매너**:
- **반드시 새 봉투** + **새 지폐** (新札)
- 봉투 표면에 받는 사람 이름, 뒷면에 주는 사람 이름
- 지폐는 인물 면이 위로 + 3단 접기 (왼쪽이 위로 가게)
- **상대 부모 앞에서 직접 전달은 결례** (사적 관계라 봉투에 넣어 전달)

**중요한 한국 차이**:
- 한국: 세배 후 받음 (쌍방향 의례)
- 일본: 어른이 일방적으로 줌 (세배 없음)
- 한국: 현금 직접 또는 봉투
- 일본: ぽち袋 (전용 봉투) 거의 필수

### 5. 初詣 (하츠모우데) — 사실 검증

**개념**: 새해 처음 신사·절 참배

**역사**:
- 起源: 江戸時代 "年籠り(도시코모리)" — 가장이 그믐~정월에 씨신(氏神)께 기도
- 그믐참배 + 정초참배로 분화
- **현재 형태**: 메이지 시대 (1868~) 중기, **京成電鐵** 등 철도회사가 참배객 수송 시작하면서 대중화

**현재**:
- 1월 1일 0시~3일 사이 가장 붐빔
- 三が日에 못 가도 松の内 안이면 OK
- **유명 신사 인파 (참고)**:
  - 메이지신궁 (도쿄): 약 300만명/3일
  - 가와사키대시 (가나가와): 약 300만명/3일
  - 나리타산 신쇼지 (지바): 약 280만명/3일
  - 후시미이나리 (교토): 약 250만명/3일

**お賽銭 (사이센, 헌금)**:
- 5엔 (ご縁, 인연) 가장 길운
- 50엔 (五重のご縁) 또는 45엔 (始終ご縁)
- 10엔은 피함 (遠縁, 멀어지는 인연으로 들림)
- 외국인은 그냥 100엔이나 500엔도 OK

**참배 순서 (二礼二拍手一礼)**:
1. 모자·후드 벗기, 짐을 내려놓음
2. 鈴 (종)이 있으면 울림
3. 사이센 投入 (헌금 던져 넣기)
4. 깊이 절 2번
5. 박수 2번 (神社만, 절은 안 함)
6. 마음 속으로 기도
7. 깊이 절 1번

**한국 학습자가 가장 헷갈리는 것**: 절은 박수 안 침. 신사만 박수.

### 6. 한국 설날과의 차이 (학습자 관점 핵심)

| 항목 | 한국 설날 | 일본 お正月 |
|---|---|---|
| 날짜 | 음력 1월 1일 (보통 양력 2월) | 양력 1월 1일 |
| 기간 | 1-3일 (보통 3일 휴일) | 三が日 또는 松の内 |
| 핵심 음식 | 떡국 (한 그릇 = 한 살) | おせち + お雑煮 |
| 의례 | 차례 (조상 제사) + 세배 | 初詣 (신사 참배) |
| 어른께 인사 | 세배 (큰절) | 없음 (그냥 おめでとう) |
| 어린이 보상 | 세뱃돈 (세배 후 받음) | お年玉 (그냥 받음) |
| 친척 모임 | 매우 중요, 큰 행사 | 三が日은 가족 단위 (외부 방문 자제) |
| 인사말 | "새해 복 많이 받으세요" | "明けましておめでとうございます" |

## 실전 표현 후보 (3개 선정 예정)

EP 본문에는 3개만 들어감. 후보 5개 중 9월 작성 시 선택.

### 후보 1: 새해 인사 (가장 기본)

**일반 격식 (직장·이웃):**
> **「明けましておめでとうございます。今年もよろしくお願いいたします」**

**친구 사이 (반말):**
> **「あけましておめでとう! 今年もよろしくね」**

→ "明ける(아케루)" = 밤이 새다, 어둠이 걷히다. 한국어 "새해"의 "새"와 통하는 어감.

**중요**: 1월 7일까지(松の内) 사용. 그 후엔 "寒中見舞い (간추 미마이)"로 전환.

### 후보 2: 初詣 권유 / 함께 가기

**격식:**
> **「初詣、一緒にいかがですか?」**
> *(하츠모우데, 같이 가시지 않을래요?)*

**친구:**
> **「初詣行こうよ! どこの神社がいい?」**
> *(하츠모우데 가자! 어느 신사 좋아?)*

→ "初詣"는 "하츠모우데"라고 정확히 읽기. "하츠모우데를 가다(参る)"는 "初詣に行く"가 일반적.

### 후보 3: お年玉 받았을 때 인사

**아이 → 어른:**
> **「ありがとうございます! 大切に使います」**
> *(감사합니다! 소중히 쓸게요)*

**부모 → 자녀에게 어른께 인사 시키기:**
> **「ちゃんとお礼を言いなさい — おじちゃん/おばちゃんに、ありがとうございますって」**
> *(제대로 인사 드려 — 삼촌/숙모께 감사합니다라고)*

→ ぽち袋를 받으면 그 자리에서 열지 않는 게 매너. 집에 가서 열어봄.

### 후보 4: おせち 관련 일상 대화

> **「おせち、自分で作りました? それとも通販?」**
> *(오세치, 직접 만드셨어요? 아니면 통판?)*

→ 동료·이웃과의 가벼운 대화. 직접 만든다 답하면 존경 받음. 통판도 일반적.

### 후보 5: 三が日 인사 자제

> **「三が日は、ゆっくりお過ごしください」**
> *(삼가니치 동안은 푹 쉬세요)*

→ 정월 3일 동안은 일반적으로 휴식 권장. 직장 동료에게 12월 30일경 보내는 인사.

## 한국어 니모닉 후보

### 후보 A (정월 = 正月 한자 매칭)

> **正月(しょうがつ)** — "정월"의 한자 그대로!
> 한국 한자어 "정월(正月)"과 1:1 매칭.
> 다만 한국어 "정월"은 음력 1월, 일본 正月는 양력 1월.

### 후보 B (おせち = 御節)

> **おせち(御節)** = "절기 음식"
> 御(어)+節(절) = 신께 바치는 절기 음식
> 한국어 "절기"라는 단어가 익숙하면 직관적.
> 평안시대 五節句(오셋쿠) 의식에서 유래.

### 후보 C (お年玉 = 年+玉)

> **お年玉(おとしだま)** = "年(년)+玉(구슬)"
> 直譯하면 "한 해의 구슬" = 새해 선물.
> 한자만 보면 한국 사람도 의미 파악 가능.

## 9월 W18 본문 작성 시 진행 방식

EP-04 회람판과 동일한 5단계 구조. 본 메모는 자료. 작성 흐름:

1. **씬 도입 (10분)** — "도쿄에서 첫 정월을 맞이한 한국 사람의 당황" 시나리오. 신사 앞에서 사이센 얼마 넣어야 할지, おせち가 뭔지, お年玉 줄 때 ぽち袋 어디서 사야 할지 등에서 1개 선정.

2. **오바아찬 스토리 (70분)** — 본 메모의 §1 시간 구조 + §2 年神様 + §3 おせち + §5 初詣 융합. ASCII 다이어그램으로 정월 시간 구조 시각화 권장 (12/13 → 12/28 → 1/1 → 1/7).

3. **실전 표현 3개 (25분)** — 후보 5개 중 3개 선정. 권장 조합: 후보 1 (새해 인사) + 후보 2 (初詣) + 후보 3 (お年玉) — 외국인 거주자 빈도 최고.

4. **깊이 들어가기 (25분)** — §6 한일 차이 표 활용. "왜 三が日은 친척 방문 자제?" + "5엔 사이센의 의미" 같은 흥미 포인트.

5. **니모닉 (5분)** — 후보 3개 중 가장 직관적인 것 선택. **권장: 후보 A (正月)** — 한국 한자어와 1:1 매칭이 가장 강력.

6. **메타 정리 + 사실 검증 체크리스트 (15분)** — 본 메모의 출처 인용.

**예상 작성 시간**: 자료가 이미 있으므로 9월에는 150-180분 (EP-04의 220분 대비 30% 절감 효과)
