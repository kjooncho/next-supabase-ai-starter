# My Niche — Design System Spec v1.0

**디자인 시스템 시트(Image 8 / 01.png)를 정본으로**, 8장 UI 화면에서 약간씩 다른 부분들을 통일하기 위한 정본 사양 문서.

본 문서와 `DESIGN_P0_GUIDE.md`의 차이:
- **`DESIGN_P0_GUIDE.md`**: PRD 4.1에서 결정됐지만 디자인 시스템 시트에 반영 안 된 변경사항(달마, 4탭 통일 등)의 *수정 행동 지침*
- **`DESIGN_SYSTEM_SPEC.md`** (본 문서): 디자인 시스템 시트 자체의 *정본 사양*, Figma에서 8장 화면을 업데이트할 때의 참조점

## ② 타이포그래피 (Noto Sans KR / JP)

| 단계 | weight | size | 용도 |
|---|---|---|---|
| H1 | Bold | 24px | 화면 메인 타이틀 (예: "My Deck", "오늘의 일본어를 배워볼까요?") |
| H2 | SemiBold | 20px | 섹션 헤더 (예: "자연스러운 표현이에요") |
| Body Medium | Medium | 16px | 본문 강조 (예: "일본에서는 이렇게 말해요.") |
| Body Regular | Regular | 14px | 일반 본문 (예: "상황에 따라 뉘앙스가 달라져요.") |
| Caption | Regular | 13px | 캡션 (예: "예시·문법·힌트") |

**일본어/숫자 별도 설정:**
- 일본어 표시: Noto Serif JP (display 클래스)
- 숫자 (Tabular Figures): font-variant-numeric: tabular-nums (요일·시간·금액 정렬용)

## ④ 아이콘 시스템

### 메인 아이콘 (24px stroke)
- 채팅 / 캘린더 / 지도·핀 / 카드덱 / 할머니 / 연속학습 / 마스터리 / 써봤어요 / 니모닉

- 🛒 가게 (Accent) / 🏥 병원 (Success) / 🏛 시청 (Primary) / 💴 은행 (Cultural Tip) / 👶 유치원 (Accent)

## ⑤ 버튼 시스템 (5종)

`<Button variant="..." size="md">레이블</Button>` 컴포넌트로 통합. (prototype.html 참조)

| variant | 배경 | 텍스트 | 외곽선 | 용도 |
|---|---|---|---|---|
| `primary` | `#e8541a` | `#fff` | none | 메인 CTA ("확인", "전송") |
| `secondary` | `#fff` | `#e8541a` | `#e8541a` | 보조 ("취소") |
| `ghost` | transparent | `#1a1a1a` | none | 텍스트 링크 ("더보기 ›") |
| `danger` | `#fff` | `#e74c3c` | `#e74c3c` | 삭제 ("삭제") |
| `realUse` | `#27ae60` | `#fff` | none | 실전 사용 기록 ("✓ 써봤어요") |

`size`: sm (12px) / md (13px, 기본) / lg (14px)

**사용 원칙:**
- 한 화면에 Primary는 1개만 (시각 우선순위)
- realUse는 카드 상세 화면의 "써봤어요" 토글 전용
- Danger는 "삭제" 외에 거의 사용 안 함
- Ghost는 inline 액션 (카드 더보기, 페이지네이션 등)

## ⑥ 카드 시스템 (5종)

각 카드는 고유한 시각 정체성을 가짐. 사용자가 한눈에 카드 종류를 구분할 수 있어야 함.

### 1. 번역 카드 (Translation)
- 배경: `#fff`
- 좌측 상단: 북마크 아이콘
- 우측 상단: 桜(벚꽃) 도장 (선택)
- 큰 일본어 본문 (Noto Serif JP)
- 한국어 의미 (작게)
- 하단: 단일 태그 (예: #ほうです)

### 2. 문화 카드 (Cultural)
- 배경: `#fef6ec` (Cultural Tip의 옅은 tint)
- 상단: 💡 + "문화" 라벨 (`#f5a623`)
- 본문: 한국어 설명 (일본 문화 노트)
- 하단: #문화표현 태그

### 3. 캘린더 카드 (Calendar)
- 배경: `#fff`
- 상단 좌측: 날짜 (큰 숫자) + 요일
- 본문: 일정/이벤트명
- 위치 정보 (선택)
- 하단: #일정 태그

### 4. 니모닉 카드 (Mnemonic)
- 배경: `#f0e9f4` (Mnemonic의 옅은 tint)
- 상단: 💡 + "니모닉" 라벨 (`#9b59b6`)
- 본문: 한자 + 어원 스토리
- 하단: #니모닉 태그

### 5. 써봤어요 카드 (Real Use)
- 배경: `#fff` 또는 살짝 녹색 tint
- 상단: ✓ "써봤어요" 라벨 (`#27ae60`)
- 본문: 사용한 표현
- 사용 일자 + 메모 (작게)

## ⑧ 캐릭터 시스템 (시트 정본 → P0 가이드 적용)

### AI-Nichi (채팅탭 메인 캐릭터)

**시트 원본**: 흰색 + 검은 무늬 둥근 얼굴 (팬더 형태)

**⚠ P0 변경**: `DESIGN_P0_GUIDE.md`의 결정에 따라 **달마(達磨)** 로 교체.
- 이유: 일본 문화 정체성 강화 (七転び八起き 학습 철학과 매핑)

- 컬러: 몸 `#e8541a`, 외곽선 `#1a1f3d`, 얼굴 `#faf8f4`

**Figma 작업 시**: 시트의 팬더를 달마로 교체 + 8장 화면 모두 동일 적용 필요

### AI-Grandma (생활탭 캐릭터)

**시트 원본 그대로 유지**:
- 회색 단발 머리
- 둥근 안경
- 따뜻한 베이지/연한 분홍 의상
- 부드러운 미소

**용도**: 隣のおばあちゃんトーク 에피소드의 화자, 생활탭 인트로

## ⑨-2 시즌 배너 (Season Banner)

**Image 7 사양**: 채팅·달력 화면 상단의 작은 pill 형태 (큰 vermillion 박스 X)

- 배경: `rgba(107, 143, 113, 0.12)` (Sage Green 옅은 tint)
- 외곽선: `0.5px solid rgba(107, 143, 113, 0.3)`
- 텍스트 색상: `var(--success)` `#6b8f71`
- 형태: rounded-full (pill)
- 구조: `🌿 이번 주 | <일본어 이벤트명> 다가옵니다 → 표현 배우기` + 우측 원형 화살표 버튼

**왜 pill인가?** 메인 화면 상단 공간을 적게 차지하면서도 매주 새 콘텐츠로 사용자를 끌어들이는 진입점. 큰 배너는 1회성 이벤트(런칭, 캠페인)에만 사용.

## ⑨-3 STEP 진행 표시 (Loading State)

번역 진행 중 표시. Image 7의 `STEP 1 ✓ → STEP 2 ✓ → STEP 3 진행 중...` 패턴.

- AI-Nichi 아바타 (mood: thinking) + 채팅 버블
- 완료된 STEP: `var(--success)` `#6b8f71` + ✓
- 진행 중 STEP: `var(--brand-accent)` `#e8541a` + 깜빡이는 점 3개
- 본문 아래 작은 캡션: "번역 결과를 준비하고 있어요"

목적: 단순 spinner보다 "지금 무엇을 하고 있는지" 명시 → 5단계 시스템에 대한 사용자 이해 강화.

## ⑨-4 최근 저장 카드 가로 스크롤

채팅 화면 하단(입력창 위)의 가로 스크롤 캐러셀. Image 7 사양.

- 라벨: `✨ 최근 저장` (uppercase tracking-wider)
- 카드 너비: `w-32` (128px), 가변 height
- 카드 내용: 일본어 표현(truncate) + 첫 번째 태그 + Mastery Badge
- 5개까지 표시 (`cards.slice(0, 5)`)
- 표시 조건: 카드가 1개 이상 있고 + greeting 외 메시지가 있을 때

목적: 학습 흐름의 연속성 — 방금 저장한 카드를 곧바로 다시 만나서 노출 빈도 증가.
