# 베타 리뷰어 카톡 메시지 템플릿 v1.0

**작성일**: 2026-05-10
**대상**: 도쿄 거주 친구 3명 (반말, 카톡)
**용도**: 베타 리뷰어 섭외 + EP 검수 안내
**발송 시점**: W2 시작 (5/18 월) — 응답 5-7일 받고 W3 (5/25-) 첫 검수 가능

---

## 사용 방법

1. 아래 메시지 3종 중 본인 상황에 맞는 거 골라서 카톡으로 발송
2. 답장 받으면 → "EP-04 안내 메시지" 발송
3. EP-04 검수 결과 받으면 → "EP-01 (お正月) 발송" 단계로

발송 전 **본인 입장에서 한 번 소리 내서 읽어보세요.** 어색한 표현 있으면 수정. 친구마다 톤 미세 조정 권장.

## 메시지 2 — 답장 받은 후 (EP-04 회람판 발송 안내)

긍정 답장 ("좋아", "해볼게" 등) 받았을 때 보내는 후속 메시지.

```
[이름]아 답장 고마워!

그럼 첫 번째 편 보낼게
"회람판 (回覧板)이 뭐죠?" 편이야

[파일 첨부 또는 링크]

체크 부탁드리는 것 (편하게):
1. 사실 정보 — 회람판 역사나 방식 설명이 맞아? 틀린 부분 있어?
2. 한국어 표현 — 자연스러워? 어색한 데 있어?
3. 일본어 표현 (3개) — 실제로 쓰면 맞는 표현이야? 너무 격식 있어?
4. 외국인 가입 절차 — 2026년 현재 맞는 정보야?
5. 그 외 "이건 좀..." 싶은 부분 자유롭게

답장 1주일 안에 주면 너무 좋고
바쁘면 그냥 "받았어" 한 마디만 줘도 OK!
```

- 한 친구 응답 보고 다른 친구 메시지 미세 조정 가능
- "여러 명에게 동시에 부탁하는 느낌" 안 들게

## 자주 받는 질문 답변 준비 (FAQ)

친구가 물어볼 가능성 있는 질문 + 답변 미리:

**Q: "그 앱 뭐야? 어떻게 생긴 건데?"**
A: 한국어로 "오늘 뭐 먹고 싶다" 같은 거 입력하면 일본어 5단계로 풀어주는 앱. 그냥 번역기랑 다른 게, 한국 사람이 자주 틀리는 부분 (가족 호칭, 경어 등) 자동 체크해주고, 회람판 같은 일본 생활 문화도 알려줘. 출시는 11월쯤이야.

**Q: "내가 일본어 잘 못하는데 괜찮아?"**
A: 오히려 좋아! 콘텐츠 자체는 한국어로 읽고, 일본어 표현이 자연스러운지만 봐주면 돼. 일본 살면서 한국 사람으로서 느낀 거 (이거 한국 사람한테 도움될 듯, 이건 한국에선 헷갈리겠다 등) 의견이 핵심.

**Q: "5편 다 검수해야 해? 1-2편만 하면 안 돼?"**
A: 1-2편만 해줘도 진짜 도움 돼. 처음 1편 받아보고 "이건 부담스럽다" 싶으면 그냥 솔직히 말해줘. 친구 관계가 우선이야.

**Q: "보상 안 받아도 돼"**
A: 아니야 진짜 시간 들이는 거니까 받아줘 ㅋㅋ 다음에 도쿄 가서 만나면 밥/술 사는 정도. 그냥 친구 사이의 답례야.

**Q: "지금 한국에 있을 때 카톡으로 보내달라는 거지?"**
A: 응! 카톡으로 PDF나 마크다운 보낼게. 모바일로 읽어도 OK. 답장도 카톡으로 자유롭게.
